2、hidden

隐藏元素时，我过去经常用 myElement.style.display = ‘none’这种方法，其实可以只需调用 myElement.hidden = true ，即可实现元素隐藏的功能。

3、toggle()

通常情况下，为元素添加或删除某个 class，我们会使用if判断语句，但是还有另外更简洁的做法——myElement.classList.toggle(‘some-class’)。
正确的方式是为 toggle 方法传入第二个参数，如果该参数返回 true ，则指定的 class 就会添加至元素上。
