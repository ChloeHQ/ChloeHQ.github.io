JS对象方法——toString()和valueOf()

一、toString()

toString()函数的作用是返回object的字符串表示，默认为形似”[object Object]”的字符串，每个实例化的对象都可以调用toString()方法。
toString()方法是定义在Object原型上的，即Object.prototype.toString().
一般，我们可以使用call,apply来调用toString方法。

1、不同类型的对象调用原型上的toString()方法

基于Object类型，JS实现了其他常用的对象子类型(例如包装对象、全局对象)：

  • Object
  • Array
  • RegExp
  • Date
  • Function
  • String
  • Boolean
  • Number
  • Error
  • Math
  • …
    不同对象都可以通过Object.prototype调用原型上的toString()方法，返回该对象的字符串表示。
2、不同的对象子类型重写的toString()方法
2.1 Array

toString():返回数组中每个元素以逗号分隔的字符串拼接。

2.2 Function

toString():返回函数的代码字符串

另外，由于Function,Array,Object等都是构造函数，因此直接用它们来调用toString()也会得到相应函数的代码。

2.3 Date

toString():返回带有时区信息的日期和时间字符串

2.4 RegExp

toString()返回正则表达式的字面量

2.5基本包装类型(Boolean, Number, String)

String
如果定义一个基本类型的字符串str，按照道理来说基本类型并没有方法可用，但是其却可以调用str.toString()，且并不会报错，这是为什么呢？
是因为由于包装类型的存在，在有必要的时候JS引擎会把基本类型的字符串字面量转化成一个String对象，这样就可以调用对应的方法。即：

  • String.prototype.toString()返回字符串字面量的一个副本
  • Number.prototype.toString()返回字符串形式的数字
  • Boolean.prototype.toString()返回字符串形式的”true”或”false”

二、valueOf()

valueOf()是一个对象方法，它返回当前对象的原始值，往往当JS需要将对象转换成某种原始值而非字符串的时候才会调用它，特别是转换成数字。

如果在需要原始值的上下文中使用了对象，JS就会自动调用valueOf()这个方法。例如，当数学运算符（+除外）和关系运算符作用于数字文本表示的对象时，会自动调用valueOf()。大多数对象没有合适的原始值表示它们，也没有定义这个方法。
